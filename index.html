git init - Comando para iniciar o versionamento de um repositorio (pasta). git add - Comando que adiciona o arquivo ou folder a ser monitorado pelo git. git rm - Remove/Desvincula um arquivo de ser traqueado do servidor git rm --cached - remove/desvincula um arquivo que ainda não foi comitado git rm -f - força a remover/desvincular um arquivo tanto do servidor como local git commit - cria um commit local git commit -m - cria um commit local já informando a mensagem (ex: git commit -m 'comentário que eu quero adicionar.') git status - exibe o status do seu repositorio local (flags -> vermelho: indica que o arquivo não foi adicionado/removido ao git [necessita do git add ou git rm] - verde: arquivos que faltam o commit)
git status - sempre mostra o status da branch atual git checkout -- arquivo - desfaz as alterações pendentes no seu arquivo
